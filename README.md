this repo help you setup new linux machine with cladue

### install claude
```
curl -fsSL https://claude.ai/install.sh | bash
```

### set claude
```
export ANTHROPIC_BASE_URL="https://ai.wuji.tech/api"
export ANTHROPIC_AUTH_TOKEN="your-api-token-here"
```
